# φ — Documentation

The user documentation for [φ](https://getpoiesis.com), a calm, local-first
writing app for manuscripts, poetry, and essays. Built with Docusaurus.

## Develop

```bash
npm install
npm start      # local dev server
npm run build  # static build into build/
```

Pages live in `docs/`. The theme (brand colors, fonts) is in
`src/css/custom.css`, matching the app's `BRAND.md`.
