# Raccourcis clavier

φ est conçu pour garder vos mains sur le clavier. Voici les raccourcis tels
qu’ils sont livrés sur macOS. Les touches de modification sont ⌘ (Commande), ⇧
(Maj), ⌥ (Option) et ⌃ (Contrôle).

Si vous en oubliez un, ouvrez la palette de commandes avec `⌘P` et commencez à
taper — la plupart des commandes affichent leur raccourci à droite.

## Général

| Action | Raccourci |
| --- | --- |
| Nouveau document | `⌘N` |
| Fermer le document | `⌘W` |
| Palette de commandes | `⌘P` |
| Rechercher | `⌘K` |
| Enregistrer | `⌘S` |
| Enregistrer une version… | `⌘⇧S` |
| Supprimer le document | `⌘⌫` |
| Ouvrir un coffre… | `⌘⇧O` |
| Réglages (Préférences) | `⌘,` |

## Édition et mise en forme

| Action | Raccourci |
| --- | --- |
| Annuler | `⌘Z` |
| Rétablir | `⌘⇧Z` |
| Couper | `⌘X` |
| Copier | `⌘C` |
| Coller | `⌘V` |
| Rechercher dans le document | `⌘F` |
| Rechercher et remplacer | `⌘⌥F` |
| Gras | `⌘B` |
| Italique | `⌘I` |
| Souligné | `⌘U` |
| Lien… | `⌘⇧K` |
| Titre 1 | `⌘⌥1` |
| Titre 2 | `⌘⌥2` |
| Titre 3 | `⌘⌥3` |
| Texte normal | `⌘⌥0` |
| Liste à puces | `⌘⇧8` |
| Liste numérotée | `⌘⇧7` |
| Citation | `⌘⇧9` |
| Aligner à gauche | `⌘⇧L` |
| Centrer | `⌘⇧E` |
| Aligner à droite | `⌘⇧R` |
| Justifier | `⌘⇧J` |

Barré et code en ligne sont dans le menu Format, sans raccourci par défaut.

## Affichage et panneaux

| Action | Raccourci |
| --- | --- |
| Afficher/masquer la barre latérale | `⌘[` |
| Afficher/masquer le panneau droit | `⌘]` |
| Dictionnaire | `⌘⇧D` |
| Mode lecture | `⌘E` |
| Mode concentration | `⌘⇧F` |
| Sanctuaire (écriture plein écran) | `⌘⇧↩` |
| Zoom avant | `⌘+` |
| Zoom arrière | `⌘-` |
| Taille réelle | `⌘0` |
| Activer/désactiver le plein écran | `⌃⌘F` |

## Navigation

Les raccourcis **Aller à** forment un accord : maintenez `⌘G`, relâchez, puis
appuyez sur la seconde touche.

| Action | Raccourci |
| --- | --- |
| Aller aux Documents | `⌘G` puis `N` |
| Aller aux Collections | `⌘G` puis `C` |
| Aller à la Recherche | `⌘G` puis `S` |
| Aller au Graphe | `⌘G` puis `G` |
