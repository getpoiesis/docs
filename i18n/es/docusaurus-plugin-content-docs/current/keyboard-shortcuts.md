# Atajos de teclado

φ está diseñado para mantener tus manos en el teclado. Estos son los atajos tal
como vienen en macOS. Las teclas modificadoras son ⌘ (Comando), ⇧ (Mayúsculas),
⌥ (Opción) y ⌃ (Control).

Si alguna vez olvidas uno, abre la paleta de comandos con `⌘P` y empieza a
escribir — la mayoría de los comandos muestran su atajo a la derecha.

## General

| Acción | Atajo |
| --- | --- |
| Nuevo documento | `⌘N` |
| Cerrar documento | `⌘W` |
| Paleta de comandos | `⌘P` |
| Buscar | `⌘K` |
| Guardar | `⌘S` |
| Guardar versión… | `⌘⇧S` |
| Eliminar documento | `⌘⌫` |
| Abrir bóveda… | `⌘⇧O` |
| Ajustes (Preferencias) | `⌘,` |

## Edición y formato

| Acción | Atajo |
| --- | --- |
| Deshacer | `⌘Z` |
| Rehacer | `⌘⇧Z` |
| Cortar | `⌘X` |
| Copiar | `⌘C` |
| Pegar | `⌘V` |
| Buscar en el documento | `⌘F` |
| Buscar y reemplazar | `⌘⌥F` |
| Negrita | `⌘B` |
| Cursiva | `⌘I` |
| Subrayado | `⌘U` |
| Enlace… | `⌘⇧K` |
| Encabezado 1 | `⌘⌥1` |
| Encabezado 2 | `⌘⌥2` |
| Encabezado 3 | `⌘⌥3` |
| Texto normal | `⌘⌥0` |
| Lista con viñetas | `⌘⇧8` |
| Lista numerada | `⌘⇧7` |
| Cita | `⌘⇧9` |
| Alinear a la izquierda | `⌘⇧L` |
| Centrar | `⌘⇧E` |
| Alinear a la derecha | `⌘⇧R` |
| Justificar | `⌘⇧J` |

El tachado y el código en línea están en el menú Formato sin un atajo
predeterminado.

## Vista y paneles

| Acción | Atajo |
| --- | --- |
| Mostrar/ocultar barra lateral | `⌘[` |
| Mostrar/ocultar panel derecho | `⌘]` |
| Diccionario | `⌘⇧D` |
| Modo lectura | `⌘E` |
| Modo concentración | `⌘⇧F` |
| Santuario (escritura a pantalla completa) | `⌘⇧↩` |

| Alejar | `⌘-` |

## Navegación

Los atajos de **Ir a** son un acorde: mantén `⌘G`, suelta, y luego pulsa la
segunda tecla.

| Acción | Atajo |
| --- | --- |
| Ir a Documentos | `⌘G` y luego `N` |
| Ir a Colecciones | `⌘G` y luego `C` |
| Ir al Diario | `⌘G` y luego `J` |
| Ir a Búsqueda | `⌘G` y luego `S` |
| Ir al Grafo | `⌘G` y luego `G` |
